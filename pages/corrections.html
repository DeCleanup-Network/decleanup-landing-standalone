FIX ITEM 07

Global text sweep — em-dashes & "onchain"

Two find/replace passes across the whole repo. Run before any other text edits so you don't reintroduce dashes.

# Pass 1 — "on-chain" → "onchain" (preserve case)
find src/ -type f \( -name "*.jsx" -o -name "*.tsx" -o -name "*.mdx" -o -name "*.json" \) -exec sed -i 's/on-chain/onchain/g; s/On-chain/Onchain/g; s/ON-CHAIN/ONCHAIN/g' {} +

# Pass 2 — em-dash audit (manual review required, context-dependent)
grep -rn " — " src/  # review each, replace with comma/period/and
grep -rn "—" src/    # catches dashes without spaces too

Don't blindly replace em-dashes. Each instance needs human judgment — see the brief's copy-corrections table for the canonical rewrites of every em-dash on the live site. Code comments and markdown tables keep their dashes.
